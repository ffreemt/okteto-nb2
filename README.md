Run/test/develop plugins for nonebot2 + gocghttp in `okteto`, agilely

## Install `okteto cli`

*   [register 注册 `okteto`](https://okteto.com/)
*   [install 安装`okteto cli`](https://okteto.com/docs/getting-started/installation/)

## Use it
*   login to 登录 `okteto`
```
okteto login
```
*   run 运行
```
okteto up
```
